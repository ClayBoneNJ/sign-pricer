# Bone Pricing Worksheet

Static pricing worksheet for sign and vehicle graphics quoting.

## Local preview

Open `dashboard.html` in a browser.

## Deployment

This repo is set up to publish with GitHub Pages from the `main` branch root.

## Files

- `dashboard.html` - app UI
- `pricing.js` - pricing logic
- `pricing-input-checklist.md` - pricing assumptions and source worksheet
